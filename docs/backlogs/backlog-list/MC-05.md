# Mã Backlog: MC-05 – Quản lý Thuốc và mã bệnh ICD-10 (Admin)

## 1. Tổng quan & Bối cảnh (Overview & Context)

### Vấn đề (Problem)
Bác sĩ thường kê đơn dựa trên thói quen, nếu không có danh mục thuốc chuẩn, việc quản lý kho và in đơn thuốc sẽ bị sai lệch đơn vị (viên/vỉ/chai). Tương tự, chẩn đoán bệnh cần theo mã ICD-10 để thống nhất y khoa toàn cầu.

### Giá trị Nghiệp vụ (Business Value)
- **An toàn**: Tránh nhầm lẫn tên thuốc gần giống nhau.
- **Số hóa**: Giúp việc thống ôn dịch tễ (các loại bệnh phổ biến) chính xác tuyệt đối.

### User Story Statement
> Là một Admin, tôi muốn thiết lập danh mục thuốc và mã bệnh lý chuẩn, để cung cấp dữ liệu chính xác cho bác sĩ trong quá trình kê đơn.

---

## 2. Luồng Người dùng (User Flow)
1. Admin vào menu "Danh mục Y tế".
2. **Thêm thuốc**: Nhập tên thuốc, nồng độ, đơn vị tính (Viên/Lọ/Gói).
3. **Thêm mã bệnh**: Nhập mã ICD-10 và tên bệnh tương ứng (Vd: E11 - Tiểu đường tuýp 2).
4. **Hỗ trợ Import**: Hệ thống hỗ trợ Import dữ liệu hàng loạt từ file CSV/Excel để tiết kiệm thời gian.

---

## 3. Tiêu chí Chấp nhận (Acceptance Criteria)
- **AC-01**: Mã bệnh ICD-10 phải là duy nhất (Unique) trong hệ thống.
- **AC-02**: Danh mục thuốc phải có bộ lọc tìm kiếm nhanh theo tên thuốc.

---

## 4. Đặc tả dữ liệu

| Tên trường | Kiểu dữ liệu | Ràng buộc | Ghi chú |
| :--- | :--- | :--- | :--- |
| **icd10_code** | String | PK, Unique | Mã bệnh chuẩn (Vd: K29, E11). |
| **disease_name** | String | Not Null | Tên bệnh tương ứng với mã. |
| **medicine_name** | String | Not Null | Tên thuốc (Vd: Paracetamol). |
| **unit** | String | Enum | Viên, Gói, Lọ... |
