# Mã Backlog: MC-02 – Thiết lập cấu trúc dự án Frontend (Next.js) và Backend (Python/FastAPI).
## 1. Mô tả chi tiết & Yêu cầu kỹ thuật
**Mục tiêu:** Khởi tạo cấu trúc nền tảng cho dự án bao gồm Frontend (Next.js) và Backend (FastAPI). Đảm bảo môi trường phát triển đồng nhất cho nhóm 4 người, thiết lập quy chuẩn cấu trúc thư mục phân lớp (Layered Architecture) để dễ dàng bảo trì, mở rộng và tránh xung đột khi gộp mã nguồn (merge code).
### Thư viện / Công nghệ khuyến nghị
- **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS, Ant Design (UI Kit), Axios.
- **Backend**: Python 3.10+, FastAPI, Uvicorn (ASGI Server).
- **Database ORM**: SQLAlchemy hoặc SQLModel.
- **Quản lý môi trường**: python-dotenv, requirements.txt (hoặc Poetry/Pipenv).
- **Code Quality (Khuyến nghị thêm)**: ESLint/Prettier (Frontend), Black/Flake8 hoặc Ruff (Backend).
### Yêu cầu kỹ thuật cốt lõi
- **Backend Architecture (Layered Pattern)**: Cấu trúc thư mục /backend phải tuân thủ:
/app/api/v1: Chứa các router, định nghĩa endpoint (nhận request/trả response).
/app/services: Chứa logic nghiệp vụ tập trung (Business Logic).
/app/crud: Chứa các hàm tương tác trực tiếp với Database (Create, Read, Update, Delete).
/app/models: Định nghĩa các thực thể bảng (Schemas) bằng SQLAlchemy/SQLModel.
/app/core: Chứa cấu hình hệ thống (config, bảo mật, biến môi trường).
- **Frontend Architecture**: Cấu trúc thư mục /frontend (Next.js):
/src/app: Sử dụng App Router cho routing.
/src/components: Chứa các UI components dùng chung (Ant Design wrappers).
/src/services: Chứa các hàm gọi API (Axios instances).
/src/types: Định nghĩa TypeScript interfaces/types.
Kết nối & Bảo mật (CORS):
Cấu hình CORSMiddleware tại file main.py để cho phép Frontend truy cập tài nguyên từ Backend.
Thiết lập file .env.example mẫu cho cả hai phía để quản lý biến môi trường (Database URL, API Key).
## 2. Tiêu chí hoàn thành (Definition of Done - DoD)
Khởi tạo thành công Git Repository với cấu trúc thư mục phân tách rõ ràng /frontend và /backend.
- **Backend**: Khởi tạo môi trường ảo, cài đặt đủ thư viện và chạy thành công lệnh uvicorn main:app --reload.
- **Frontend**: Khởi tạo dự án Next.js (TypeScript) chạy thành công lệnh npm run dev hoặc yarn dev.
- **Kết nối**: Endpoint /health-check (hoặc /) trả về JSON thành công khi gọi từ trình duyệt hoặc Postman.
- **CORS**: Frontend thực hiện gọi thử một API từ Backend thành công mà không bị lỗi Cross-Origin.
- **Tài liệu**: Có tệp README.md hướng dẫn chi tiết các bước cài đặt môi trường (Setup guide) để cả 4 thành viên đều có thể chạy dự án trên máy cá nhân trong < 15 phút.
Đồng bộ hóa: Có đầy đủ file requirements.txt (Backend) và package.json (Frontend) đã được cập nhật phiên bản thư viện mới nhất.
