# Mã Backlog: MC-02 – Thiết lập cấu trúc dự án Frontend (React.js) và Backend (Python/FastAPI)

## 1. Tổng quan & Bối cảnh (Overview & Context)

### Vấn đề (Problem)
Khi làm việc nhóm 4 người, nếu mỗi thành viên code theo phong cách riêng hoặc cấu trúc thư mục lộn xộn, việc gộp code (merge) sẽ cực kỳ khó khăn và dễ gây ra lỗi hệ thống nghiêm trọng.

### Giá trị Nghiệp vụ (Business Value)
- **Tính chuyên nghiệp**: Dễ bảo trì và mở rộng tính năng trong tương lai.
- **Tốc độ**: Giúp Frontend và Backend phối hợp nhịp nhàng thông qua các chuẩn giao tiếp đã định sẵn (RESTful API).

---

## 2. Luồng xử lý (Technical Flow)
1. **Backend**: Khởi tạo môi trường ảo (venv), cài đặt FastAPI và Uvicorn. Cấu hình dự án theo kiến trúc phân lớp:
   - `api/v1`: Định nghĩa các Router (nhận request).
   - `services`: Xử lý logic nghiệp vụ tập trung.
   - `crud`: Thực hiện các câu lệnh truy vấn Database.
   - `models`: Định nghĩa các thực thể bảng bằng SQLAlchemy/SQLModel.
2. **Frontend**: Khởi tạo dự án React.js bằng Vite để tối ưu tốc độ build. Cài đặt các thư viện cốt lõi: Ant Design (UI Kit), Axios (gọi API) và Tailwind CSS.
3. **Kết nối (CORS)**: Cấu hình `CORSMiddleware` trong file `main.py` của FastAPI để cho phép ứng dụng Frontend truy cập và trao đổi tài nguyên với Backend.
4. **Kiểm tra**: Tạo một Endpoint đơn giản (ví dụ: `/health-check`) trả về JSON thành công để xác nhận luồng dữ liệu từ Frontend đến Backend và ngược lại đã thông suốt.
