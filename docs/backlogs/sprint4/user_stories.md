# TUẦN 4: SPRINT 4 – TRÍ TUỆ NHÂN TẠO & HOÀN THIỆN (POLISHING)

## 1. Mục tiêu (Sprint Goal)
Tích hợp các tính năng AI thông minh nâng cao và đóng gói sản phẩm hoàn thiện.

## 2. Backlog triển khai
- **MC-10**: Kê đơn thuốc và xuất bản file PDF kết quả khám.
- **MC-11**: Bệnh nhân xem Timeline lịch sử và tải hồ sơ.

## 3. Kết quả mong đợi (Sprint Delivery)
Hệ thống hoàn thiện 100% tính năng, tích hợp trợ lý AI, xuất bản đơn thuốc định dạng PDF và giao diện báo cáo thống kê trực quan.
